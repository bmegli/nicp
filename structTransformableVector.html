Detailed Description

template<typename Transformable>
struct TransformableVector< Transformable >

Base class for the creation and manipulation of vectors of objects.

This class allows to create and manipulate a vector of object where the type of the objects composing the vector is specified inside the template. The only manipulation allowed is the transformation of all obejcts composing the vector by means of a matrix multiplication.

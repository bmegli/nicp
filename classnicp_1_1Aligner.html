Detailed Description

Class for point cloud alignment.

This class is an interface for objects that want to align two point clouds.

This class allows to compute the homogeneous transformation that brings a point cloud to superpose an other reference point cloud. Data association is computed via nearest neighbor.

This class allows to compute the homogeneous transformation that brings a point cloud to superpose an other reference point cloud. Data association is computed via depth image projection.

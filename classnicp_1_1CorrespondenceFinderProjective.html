Detailed Description

Class that can be used to find correspondences between two reprojected point clouds.

This class has the objective to find correspondences between two point clouds reprojected into their depth images. A correspondence has to satisfy some constraint like maximum angle between normals, maximum distance between the points and so on. The algorithm just pick points in the same position inside the repsective depth images and make a comparison to see if the two points are a correspondence, if they satisfy all the constraints then the pair of points is added to the vector of correspondences.

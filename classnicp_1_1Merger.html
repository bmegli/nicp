Detailed Description

Class for point cloud merging.

This class allows to merge points in a point cloud. This is particularly useful when multiple point clouds are added togheter in a single one. This creates an excess of points that are not necessary and slow the computation. The merging process will reduct the points erasing the ones in surplus.
